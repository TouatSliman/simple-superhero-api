This is a simple super heros and super villains api.

somme links : GET

  • https://simple-superhero-api-sba.onrender:4040/ that give you all json code
  • https://simple-superhero-api-sba.onrender/names that gives you all names
  • https://simple-superhero-api-sba.onrender/names&id gives you all names and id's
  • https://simple-superhero-api-sba.onrender/superheros/ gives you all super heros
  • https://simple-superhero-api-sba.onrender/superheros/names gives you all super heros names
  • https://simple-superhero-api-sba.onrender/superheros/names&id gives you all super heros names and id
  • https://simple-superhero-api-sba.onrender/superheros/3 replace 3 with any id you want
  • https://simple-superhero-api-sba.onrender/supervillains/ that gives you all villains
  • https://simple-superhero-api-sba.onrender/supervillains/names that gives you all villains names
  • https://simple-superhero-api-sba.onrender/supervillains/names&id that gives you all villains names and id
  • https://simple-superhero-api-sba.onrender/supervillains/3 replace 3 with any id you want
